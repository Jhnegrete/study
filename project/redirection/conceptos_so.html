Conceptos Basicos de Sistemas Operativos



Que es un Sistema Operativo

Conjunto de órdenes y programas que controlan los procesos básicos de una computadora y permiten el funcionamiento de otros programas. En palabras poco recomendadas es el programa encargado de controlar todos los programas que se ejecutan en un computador.

O tanbien se podria decir que un Sistema Operativo (SO) es el software básico de una computadora que provee una interfaz entre elresto de programas del ordenador, los dispositivos hardware y el usuario.

Las funciones básicas del Sistema Operativo son administrar los recursos de la máquina, coordinar el hardware y organizar archivos y directorios en dispositivos de almacenamiento.

Los Sistemas Operativos más utilizados son Dos, Windows, Linux y Mac. Algunos SO ya vienen con un navegador integrado, como Windows que trae el navegador Internet Explore

Para que sirve

El sistema operativo es el intermediario entre el usuario y el hardware, antes el usuario tenia que interactuar directamente con el hardware pero ahora lo hace por medio del SO.
Entonces se podria decir que el sistema operativo facilita la experiencia del usuario.

Como funcionan

Los sistemas operativos proporcionan una plataforma de software encima de la cual otros programas, llamados aplicaciones, puedan funcionar. Las aplicaciones se programan para que funcionen encima de un sistema operativo particular, por tanto, la elección del sistema operativo determina en gran medida las aplicaciones que puedes utilizar.

Historia de los Sistemas Operativos

A media que ha pasado el tiempo los Sistemas Operativos han evolucionado de una forma impresionante.

Sistemas Operativos por lotes

  • Tipo de carga: Trabajos con secuencia de ráfagas de CPU y E/S. La diferencia de velocidad entre la CPU y la E/S es de tres órdenes de magnitud o más.
  • Utilizacion de la CPU: tiempo_CPU_ocupada / tiempo_total.
  • Ausencia de interacción usuario - máquina.
  • Agrupación por lotes de trabajos similares
  • Secuenciación automática trabajos con tarjetas de control

Spoling

Buscan reducir los tiempos de E/S utilizando el disco como un buffer para compensar la diferencia de velocidades entre la E/S y los periféricos lentos.

  • El disco actúa como un buffer de E/S.
  • Mientras el periférico está ocupado, la CPU vuelca a disco.
  • Cuando el periférico está listo, se realiza la transferencia disco - periférico .

Sistemas operativos Multiprogramados

Ellos aprovechar las ráfagas de espera (de la E/S) de un trabajo para ejecutar las ráfagas de CPU de otros trabajos.
Son sistemas por lotes en los que se introduce la técnica de multiprogramación.
En los sistemas por lotes sin spooling, no podía haber multiprogramación puesto que se dependía del lector de tarjetas. No se podía intercalar la ejecución de dos trabajos diferentes, puesto que sólo las tarjetas de uno de ellos podían estar en el lecto

Sistemas de tiempo compartido

Varios procesos se ejecuatan al mismo tiempo, con el tiempo compartido se inicia la interaccion del usuario con la maquina, porque antes en los Sistemas por Lotes el usuario llegaba introducia la cinta, luego el sistema la leia y agrupapa los procesos por lotes y el usuario no volvia a interactuar con el sistema hasta que se terminara de ejecutar el programa.

Ahora hay varios procesos ejecutandoce al mismo tiempo y se mejora el tiempo de respuesta por lo tanto el usuario puede estar al frente indicando algo y recibiendo respuestas (rapadas);

  • Tiempos de respuesta accesible
  • Mejora la interfas Usuario - Maquina
  • Multitarea: un usuario puede ejecutar varios procesos concurrentemente.
