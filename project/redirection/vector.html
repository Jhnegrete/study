Vector

Un vector en una estructura que almacena una serie de datos uno seguido del otro, es decir en forma secuencial, los datos que se almacenan deben ser del mismo tipo (int, double, string, etc), se podria decir los vectores son las estructuras mas simple y las mas usadas.
Forma en que se representa un vecor

Como se observa el tamaño del vector es 10 pero por convencion se empiza a tomar los indices desde cero, es decir el vector tiene 9 posiciones y a esas posiciones se hace referencia por medio de los indices.
El acceso a cada elemento del vector se realiza atraves de un indice, segun la posicion ocupada por el elemento dentro del vector.
Un vector siempre tiene un tamaña definido, el cual no varia a lo largo de la ejecucion del programa que lo utilice.
Estrucutura de datos

Haciendo un recuento podemos decir que un vector es una estructura de datos estatica, que posee un tamaño definido que no cambia; los datos que almacenamos en ellos deben ser del mismo tipo, la forma de acceder a un dato es por medio de su indice.

Codificando en java

Como se declara un vector en java ? 

                tipo_dato[] nombre = new tipo_dato[tamaño];
                o tipo_dato[] nombre = {dato,dato,dato};
Entonces si queremos crear un vector para almacenar 5 edades. 
                int[] edad = new int[5];
                // Se declaro pero aun no se ha llenado
Para llenarlo debemos recorrerlo mediante un ciclo 
                for(int i = 0; i < edad.length; i++){
                // con edad.length obtengo el tamaño del vector
                edad[i] = 2*i; // Lo lleno con los pares
                }
Para mostrarlo en pantalla 
                  for(int i = 0; i < edad.length; i++){
                  System.out.print(edad[i]+" ");
                }
Y el resultado sera: 
                0 2 4 6 8
